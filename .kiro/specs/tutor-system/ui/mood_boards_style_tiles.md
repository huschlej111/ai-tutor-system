# Mood Boards & Style Tiles: Know-It-All Tutor System

## Overview

This document establishes the visual direction, aesthetic principles, and design language for the Know-It-All Tutor platform. The mood boards capture the desired emotional tone while style tiles define the concrete visual elements.

## Design Philosophy

### Core Brand Attributes
- **Approachable**: Learning should feel welcoming, not intimidating
- **Intelligent**: Sophisticated without being overwhelming
- **Progressive**: Clear sense of advancement and achievement
- **Trustworthy**: Reliable platform for serious learning goals
- **Energizing**: Motivating and encouraging continued engagement

### Emotional Goals
- **Confidence**: Users feel capable of mastering complex subjects
- **Clarity**: Information is always clear and well-organized
- **Achievement**: Progress and success are celebrated meaningfully
- **Focus**: Interface supports deep learning without distractions

### Animation & Transitions

#### Timing Functions
```
Ease Out: cubic-bezier(0.0, 0.0, 0.2, 1)
Ease In: cubic-bezier(0.4, 0.0, 1, 1)
Ease In Out: cubic-bezier(0.4, 0.0, 0.2, 1)

Durations:
- Micro: 150ms (hover states)
- Short: 250ms (transitions)
- Medium: 350ms (page changes)
- Long: 500ms (complex animations)
```

#### Animation Principles
```
Purposeful: Every animation serves a function
Subtle: Enhance, don't distract from content
Consistent: Same timing and easing throughout
Accessible: Respect prefers-reduced-motion
Performance: Use transform and opacity when possible
```

#### Interactive Elements

```
Error States: Color + text + icons

```
